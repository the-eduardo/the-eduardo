# Hey, I'm Eduardo 👋

Backend developer writing mostly **Go**, with a soft spot for **Discord bots that do real ops work**, **self-hosted infrastructure**, and automating away anything manual.

- 🤖 I build Discord bots for real tasks — monitoring Docker hosts, tracking GitHub releases, managing communities
- 🏗️ I run my own infra: Docker, Grafana/Prometheus and Uptime Kuma on Oracle Cloud, with DNS, Workers and WAF on Cloudflare
- 🧠 Automating ops with LLM agents — autonomous code review, scheduled maintenance and self-healing deploys
- 🚧 Currently building: a multi-tenant Discord bot platform for trading communities, and a bid-intelligence platform for Brazilian public auctions, starting with the energy sector
- 🎮 Long-time builder of Battlefield community tooling (PunkBuster, RCON, server admin)
- 🧪 Always tinkering: CLIs, desktop utilities, custom themes — whatever scratches the itch
